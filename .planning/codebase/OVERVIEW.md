# Codebase Overview: VoxChain

## Purpose
VoxChain is a premium civic education platform designed to empower voters with transparent, ZK-verified knowledge about election systems and procedures. It combines state-of-the-art 3D visualizations with secure blockchain verification to build trust in democratic institutions.

## Core Pillars
1. **ZK-Verified Eligibility**: Real proof-of-age and non-duplicity verification using Midnight Network.
2. **AI Civic Assistant**: Grounded education via Google Gemini, providing instant answers to voting queries.
3. **Immersive Education**: Interactive 3D timelines and data visualizations powered by GSAP and React.
4. **Privacy-First Identity**: Secure authentication via Firebase and anonymous session handling.

## Stack at a Glance
- **Frontend**: Next.js 15, React 19, Tailwind CSS 4.
- **Backend**: Next.js API Routes, Google Cloud Run.
- **Data**: Firebase Auth, Firestore, Cloud Translation.
- **Web3**: Midnight SDK, Compact (ZK-DSL).
- **Animation**: GSAP (GreenSock), ScrollTrigger.

## Navigation
- **Primary**: Centered glassmorphism pill for main site traversal.
- **Utilities**: Stacked top-right sidebar for Language, Auth, and Wallet.
