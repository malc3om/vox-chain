# Stack

- **Framework**: Next.js 16.2.4 (App Router)
- **Language**: TypeScript (v5)
- **Styling**: Tailwind CSS v4, PostCSS
- **Cryptography**: `jose` (v6.2.3) for Edge-compatible JWT signing, `@noble/ed25519` for wallet signature verification.
- **Node Environment**: Edge runtime for API routes.
- **Testing**: Vitest for unit tests, Playwright for E2E testing.
