# Structure

- `src/app/`: Next.js App Router (pages and API routes).
- `src/components/`: Reusable React components (UI and layout).
- `src/lib/`: Business logic, constants, and external integrations (`ai/`, `midnight/`, `quiz/`).
- `src/types/`: TypeScript definitions (`midnight.ts`, `election.ts`).
- `contracts/`: Intended directory for raw Midnight Compact smart contracts.
- `proofs/`: Client-side logic for generating zero-knowledge proofs.
- `.planning/`: Agentive workflows tracking and generated context mapping.
