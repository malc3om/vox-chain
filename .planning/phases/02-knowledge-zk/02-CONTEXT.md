# Phase 2: Knowledge & ZK-Proofs

## Context
This phase focuses on the core functional pillars of VoxChain: real Zero-Knowledge voter eligibility verification and a grounded AI assistant. We transition from UI polish to deep integration with the Midnight Network and Gemini AI.

## Objectives
1. **Real ZK Verification**: Replace placeholder buttons with actual Compact-based ZK circuits for age and uniqueness verification.
2. **Grounded AI**: Connect the Gemini Assistant to a curated "Civic Knowledge Base" to ensure accuracy in election education.
3. **Verified Knowledge**: Implement a quiz engine where high scores are proven via ZK nullifiers (optional but preferred for gamification).

## Stakeholders
- Voters (Users): Need a seamless, privacy-preserving way to verify eligibility.
- Election Officials (Simulated): Need to trust the verification without seeing private data.
